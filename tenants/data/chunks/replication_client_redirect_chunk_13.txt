Snowflake CLI
¶
Specify the host name for the connection URL in the
account
connection parameter in the Snowflake CLI
config.toml
file. For information
about the
config.toml
file, see
Configuring Snowflake CLI
.
account = <organization_name>-<connection_name>
username = <username>
password = <password>
Copy
For example:
[connections.myconnection]
account
=
"myaccount"
user
=
"jondoe"
password
=
"password"
Copy
